# GitHub Pages Setup Instructions

This repository is ready for GitHub Pages deployment. Follow these steps to activate it:

## Activation Steps

1. Go to your repository settings: https://github.com/CtrlAltSam/handover/settings/pages
2. Under "Source", select the branch: `copilot/create-html-site-for-showcase`
3. Keep the folder as `/ (root)`
4. Click "Save"
5. Wait a few minutes for GitHub to build and deploy your site

## Accessing Your Site

Once deployed, your site will be available at:
- https://ctrlaltsam.github.io/handover/

## Pages Available

- **Homepage**: `/` or `/index.html` - Landing page showcasing the app
- **Downloads**: `/downloads.html` - Download page with version history and system requirements
- **Privacy Policy**: `/privacy.html` - Comprehensive privacy policy
- **Terms of Service**: `/terms.html` - Complete terms of service
